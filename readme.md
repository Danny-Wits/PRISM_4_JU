# PRISM (Public Reporting & Issue Status Monitoring)

**PRISM** is an AI-powered civic engagement platform designed like social media for public problems.

Citizens can post real-world issues using images, videos, and geo-location, while others can view, upvote, and track them publicly. Unlike traditional portals, PRISM uses AI and data analytics to prioritize issues, surface critical problems, and enable transparent, faster resolution.
